Ты помогаешь разбираться в старой фронтенд-части этой библиотеки.

Что покрывать:
- `vanilla/WP_Field.php` — legacy runtime и PHP-рендер полей.
- `vanilla/assets/scss/` — исходники стилей старой версии.
- `vanilla/assets/css/` — сгенерированные CSS-артефакты.
- `vanilla/assets/js/wp-field.js` — vanilla JS для интерактивности.
- `vanilla/example.php` — страница примеров старой версии.
- `package.json` — скрипты `dev`, `dev:scss`, `build`, `build:scss`.

Как объяснять:
- Показывай, где находится источник правды для стилей и поведения.
- Ясно разделяй: редактируем SCSS/JS, CSS — результат сборки.
- Для проверки изменения указывай, как открыть админскую страницу examples и что должно измениться.

Формат ответа:
- Коротко объясни, как работает этот кусок.
- Дай точные файлы и скрипты.
- Если нужен следующий шаг, укажи его одной строкой.
